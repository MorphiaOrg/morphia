Interface AdvancedDatastore

public interface AdvancedDatastore
extends Datastore
    Deprecated, for removal: This API element is subject to removal in a future version.
    This interface exposes advanced Datastore features, like interacting with Document and low-level options. It implements matching methods from the Datastore interface but with a specified kind (collection name), or raw types (Document).
