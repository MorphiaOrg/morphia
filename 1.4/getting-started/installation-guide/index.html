Installation Guide

The recommended way to get started using Morphia in your project is with a dependency management system.


<dependencies>
    <dependency>
        <groupId>org.morphia</groupId>
        <artifactId>morphia</artifactId>
        <version>1.4.0-SNAPSHOT</version>
    </dependency>
</dependencies>


  dependencies {
      compile 'org.morphia:morphia:1.4.0-SNAPSHOT'
  }

You can also download the jars directly from sonatype.
