# Features

Morphia provides a rich set of features for mapping Java objects to MongoDB documents. Explore the various capabilities below.

## Core Features

- [Configuration](/features/configuration/) - Configure Morphia for your application
- [Mapping](/features/mapping/) - Map Java classes to MongoDB documents
- [Querying](/features/queries/) - Query your data with type-safe APIs
- [Updating](/features/updates/) - Update documents in your database
- [Deleting](/features/deletes/) - Delete documents from your database

## Advanced Features

- [Aggregation](/features/aggregations/) - Use MongoDB's aggregation framework
- [Indexing](/features/indexing/) - Define and manage indexes
- [Transactions](/features/transactions/) - Work with multi-document transactions
- [References](/features/references/) - Handle document references
- [Life Cycle Methods](/features/lifecycle-methods/) - Hook into entity lifecycle events
- [Schema Validation](/features/schema-validation/) - Validate your document schemas

## Language Support

- [Kotlin](/features/kotlin/) - Kotlin-specific features and support
