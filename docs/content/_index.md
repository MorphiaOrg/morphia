# Morphia Documentation

Welcome to the Morphia documentation. Morphia is a lightweight type-safe library for mapping Java POJOs to/from MongoDB.

## Quick Links

- [Getting Started](/getting-started/) - Learn how to set up and use Morphia
- [Features](/features/) - Explore Morphia's capabilities
- [Support](/support/) - Get help and contribute

## What is Morphia?

Morphia wraps the MongoDB Java driver so some level of familiarity with using the driver can be helpful. Morphia does its best to abstract much of that away but if something is confusing, please consult the Java driver documentation as well.
